# Emotion Detector

A web application that detects your emotional state and reacts accordingly.

## Requirements

- [Vite](https://vitejs.dev/)
- [Node.js](https://nodejs.org/en/)

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

## Linter

```bash
npm run check
```

## Format

```bash
npm run format
```
